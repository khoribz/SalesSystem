## Система продажи кофе
Чтобы запустить локальный сервер, на котором будет происходит все взаимодействие с пользователем
, необходимо в терминале *linux* следующие команды:
```bash
git clone https://github.com/khoribz/SalesSystem.git
cd SalesSystem
pip3 install -r requirements.txt
python3 manage.py runserver
```  
В программе использован архитектурный паттерн Model-View-Controller (MVC), суть которого в разделении
данных приложения и управления логики на три отдельных компоненты: модель, представление и контроллер. 
Причина использования паттерна - независимость модификации каждой отдельной части.

Для проекта использован фреймворк  Django для веб-приложений на язык Python.
В выбранном фреймворке роль модели выполняют файлы **models.py**, представления - файлы папки **templates**,
контроллера - **views.py**

В данном проекте использована база данных sqlite3. Каждый сделанный заказ заносится в нее.
